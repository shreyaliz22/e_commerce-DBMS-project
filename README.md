# e_commerce-DBMS-project

Designed and implemented a MySQL database for an e-commerce platform to manage users, products, orders, payments, and reviews.
• Created stored procedures, triggers, and views for automated stock updates, reporting, and analytics.
• Applied 3NF normalisation, indexing, and transactions to ensure data integrity, performance, and scalability.
• Optimised database workflows to support secure, efficient, and consistent order processing across all modules
